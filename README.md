# astrbot_plugin_lottery_modes

## 🤝 介绍

抽奖插件增强版，在原有即时开奖基础上新增**定时抽奖模式**：
- **即时模式**：用户发送“抽”立即开奖，即开即中。
- **定时模式**：用户先报名，管理员可设置 **CRON 表达式** 实现自动开奖，也可手动开奖；开奖时按奖项等级顺序从报名者中随机抽取（每人最多中一奖），开奖后活动结束。

> 本插件基于 [astrbot_plugin_lottery](https://github.com/Zhalslar/astrbot_plugin_lottery) 进行二次开发，定时调度功能参考了 [astrbot_plugin_reminder](https://github.com/Foolllll-J/astrbot_plugin_reminder) 的实现思路。感谢两位原作者的贡献！

## 📦 安装

在 AstrBot 插件市场搜索 `astrbot_plugin_lottery_modes`，点击安装即可。

## ⌨️ 使用说明

### 配置文件

在插件配置中可设置默认抽奖模式：

```json
{
    "lottery_mode": "instant",   // "instant" 或 "scheduled"
    "default_prize_config": { ... }  // 默认奖项配置
}
```

### 命令表

| 命令 | 权限 | 说明 |
|------|------|------|
| `开启抽奖 [即时/定时]` | 管理员 | 开启抽奖活动，不指定模式时使用配置默认值 |
| `抽` | 成员 | 即时模式：立即抽奖；定时模式：报名参与 |
| `设置奖项 <等级> <概率> <数量>` | 管理员 | 动态调整特等/一等/二等/三等奖的中奖概率和总数 |
| `关闭抽奖` | 管理员 | 停止当前活动（不再接受报名/抽奖） |
| `重置抽奖` | 管理员 | 彻底删除本群抽奖数据 |
| `抽奖状态` | 成员 | 查看活动状态、参与人数、奖品剩余、是否设置自动开奖 |
| `中奖名单` | 成员 | 查看已中奖用户及对应奖项 |

#### 定时模式专用命令

| 命令 | 权限 | 说明 |
|------|------|------|
| `设置开奖cron <cron表达式>` | 管理员 | 设置自动开奖时间（例如 `设置开奖cron 0 12 * * *` 表示每天12:00开奖） |
| `取消开奖cron` | 管理员 | 取消已设置的自动开奖 |
| `开奖` | 管理员 | 立即手动开奖（不依赖定时） |

### CRON 表达式说明

使用标准 5 段表达式：`分 时 日 月 周`

| 字段 | 允许值 | 特殊字符 |
|------|--------|----------|
| 分钟 | 0-59 | `*` `,` `-` `/` |
| 小时 | 0-23 | `*` `,` `-` `/` |
| 日 | 1-31 | `*` `,` `-` `/` |
| 月 | 1-12 | `*` `,` `-` `/` |
| 星期 | 0-7 (0和7都表示周日) | `*` `,` `-` `/` |

常用示例：
- `0 12 * * *` – 每天中午12:00
- `0 9,18 * * *` – 每天9:00和18:00
- `*/30 * * * *` – 每30分钟
- `0 9 * * 1` – 每周一上午9:00

### 使用流程示例

**即时模式**：
1. 管理员：`开启抽奖 即时`
2. 成员：`抽` → 立即得知中奖结果
3. 管理员：`关闭抽奖` 结束活动

**定时模式**：
1. 管理员：`开启抽奖 定时`
2. 成员：`抽` → 报名成功
3. 管理员：`设置开奖cron 0 18 * * *` → 设置每天18:00自动开奖
4. 到达设定时间 → 系统自动开奖并发送结果到群
5. 管理员也可随时用 `开奖` 手动开奖

## 📌 注意事项

- 定时模式开奖后活动自动结束，未中奖的用户无法补抽。
- 自动开奖需要插件保持运行，重启后会自动恢复未开奖的定时任务。
